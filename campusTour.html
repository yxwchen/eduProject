\ 水土共生島 /

歷經921地震的中部在悲痛中重生，得到了與水土共生的力量，災後的重建與蛻變，讓這片土地再次展翅。

山海探索島

\ 山海探索島 /

灑滿金色陽光的南台灣，課程結合了自然與傳統文化，搭配戶外活動，讓民眾學習與大地之母和平共處。

風雨森林島

\ 風雨森林島 /

嘉南地區山坡地陡峭，沿海地勢低窪，每逢颱風季經常受到風雨攪擾，所以居民都是身經百戰的風雨勇士。

越野生存島

\ 越野生存島 /

你懂得在野外覓食、住宿，並且安全抵達集結點嗎?豐富的野外求生、急救、勘查、定向等主題等你來挑戰!

\ 祖靈傳說島 /

先人透過傳說、禁忌、習俗教我們如何崇敬自然，讓我們一起聆聽祖靈的話語，學習先人的智慧吧!

科技智慧島

\ 科技智慧島 /

自然與科技兼具的北臺灣，結合自身優勢與防災課程，互動體驗災害與避難，一起來解鎖這些課程關卡吧!

水域生態島

\ 水域生態島 /

臺灣環山面海，水資源管理與我們息息相關，一起來學習淹水檢測、海洋防災資源與海上安全準備等課程吧!
